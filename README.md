# Repo Explorer

A web application for browsing and exploring GitHub repositories with ease.

Features

- Repository Navigation: Seamlessly navigate through repository files and directories.

- File Viewer: View contents of files directly within the app.

- Search Functionality: Quickly search for files and folders within a repository.

- Responsive Design: Optimized for both desktop and mobile views.
